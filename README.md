# Federated Learning Image Classification System

A distributed image classification platform built using Federated Learning principles, allowing multiple clients to collaboratively train a machine learning model without sharing their raw image data.

Developed as part of the Software Engineering Group Project (GRP) at the University of Nottingham Ningbo China.

---

## Project Overview

Traditional machine learning systems require users to upload data to a central server for training.

This project explores a privacy-preserving alternative through Federated Learning, where multiple clients train local models on their own devices and only share model updates with a central aggregation server.

The system demonstrates how distributed machine learning can be applied while reducing data privacy risks.

↓

↓

↓
Federated Server
↓
Aggregation Process
↓
Global Model Update
↓
Redistribution to Clients
