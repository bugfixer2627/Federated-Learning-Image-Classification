# Frontend Workflow

Although the primary focus of this project was federated learning, frontend planning and user workflow design were created to demonstrate how end users would interact with the system.

## User Journey

User
↓
Access Platform
↓
Connect Client Device
↓
Local Dataset Selection
↓
Local Training
↓
Model Update Submission
↓
Federated Aggregation
↓
Updated Global Model
↓
Performance Review

## Design Goals

- Simple user interaction
- Clear visibility of training progress
- Privacy-first workflow
- Transparent model update process
